 ⏳ TelaDeCarregamento

Projeto **TelaDeCarregamento** desenvolvido com **HTML e CSS**, com o objetivo de criar uma animação de loading moderna para interfaces web. O foco do projeto é trabalhar animações, centralização e estilização visual utilizando apenas HTML e CSS.

 📌 Sobre o Projeto

O **TelaDeCarregamento** é uma interface de loading que simula o carregamento de uma página ou conteúdo. O design foi criado para ser simples, leve e reutilizável em diferentes projetos.

Este projeto foi desenvolvido para praticar:

* Estruturação com HTML
* Animações com CSS
* Centralização de elementos
* Estilização visual
* Componentes reutilizáveis

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3

## 🎯 Funcionalidades
